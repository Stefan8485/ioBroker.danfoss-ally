Danfoss Ally Cloud Adapter v0.1.0

Vielen Dank für die Installation des Danfoss Ally Cloud Adapters für ioBroker.

Der Adapter stellt eine Verbindung zur Danfoss-Cloud her und liest Temperatur, Feuchte, Ventil- und Batteriewerte aus.

  • Lizenz: GPL v3
  • Autor: Stefan Koch / Koch-CableTec
  • Website: GitHub-Repository

Nach der Installation öffne bitte die Instanzeinstellungen und trage deinen API Key und Secret ein.
